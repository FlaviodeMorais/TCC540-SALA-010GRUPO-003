Arquivos para Download

Abaixo estão os arquivos para implantação do sistema de monitoramento aquapônico desenvolvido para o TCC:

Backend (para Render.com)

Baixar TCC.BACK_complete.zip

Pacote completo para implantação do backend no Render.

Frontend (para GitHub Pages)

Baixar TCC.FRONT.zip

Arquivos básicos para implantação do frontend no GitHub Pages.

Instruções para Implantação

Para o Backend (Render):

  1. Crie uma conta em render.com se ainda não tiver
  2. Extraia o arquivo TCC.BACK_complete.zip
  3. Faça upload do conteúdo para um repositório GitHub
  4. No Render, selecione "New Web Service" e conecte ao seu repositório GitHub
  5. Configure as seguintes variáveis de ambiente: 
    THINGSPEAK_CHANNEL_ID=2886349
THINGSPEAK_READ_KEY=HKYT3SCF97WUG6OI
THINGSPEAK_WRITE_KEY=OQKOVTQQJ4I0PIPQ
  6. Defina o comando de construção: npm install
  7. Defina o comando de início: npm start
  8. Clique em "Create Web Service"

Para o Frontend (GitHub Pages):

  1. Extraia o arquivo TCC.FRONT.zip
  2. Faça upload do conteúdo para um repositório GitHub
  3. No arquivo .env.production, atualize a URL do backend para o endereço do seu serviço no Render
  4. Nas configurações do repositório, ative o GitHub Pages apontando para a branch principal
  5. Execute o comando npm run deploy para publicar o site
