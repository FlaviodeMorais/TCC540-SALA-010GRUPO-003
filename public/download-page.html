Sistema de Monitoramento Aquapônico

Obter Cópia do Sistema

Para obter uma cópia do sistema de monitoramento aquapônico que está sendo executado atualmente, você pode utilizar um dos métodos abaixo:

Método 1: Download via GitHub

Clone o repositório diretamente do GitHub utilizando o seguinte comando:

git clone https://github.com/seuusuario/aquaponia-monitor.git

Este método garante que você tenha acesso ao código-fonte completo e ao histórico de alterações.

Método 2: Download dos Arquivos Essenciais

Solicite ao administrador do sistema o arquivo aquaponia_sistema_essencial.zip que contém todos os componentes necessários.

Alternativamente, acesse o sistema no seguinte endereço para verificar se o download está disponível:

Verificar Disponibilidade de Download

Método 3: Exportação do Banco de Dados Atual

Se você precisar apenas dos dados mais recentes do sistema, solicite ao administrador uma exportação do banco de dados.

Instruções de Instalação

  1. Extraia o conteúdo do arquivo ZIP (se utilizar o Método 2) ou clone o repositório (se utilizar o Método 1)
  2. Instale as dependências com o comando: npm install
  3. Configure as variáveis de ambiente no arquivo .env utilizando os seguintes valores:
    4. THINGSPEAK_READ_API_KEY=5UWNQD21RD2A7QHG
THINGSPEAK_WRITE_API_KEY=9NG6QLIN8UXLE2AH
THINGSPEAK_CHANNEL_ID=2840207
REFRESH_INTERVAL=300000
PORT=3000
DATABASE_FILE=aquaponia.db
BACKUP_DATABASE_FILE=aquaponia_backup.db
  4. Execute o sistema com: npm run dev
  5. Acesse o sistema em seu navegador através de: http://localhost:3000

Requisitos do Sistema

  • Node.js 18 ou superior
  • NPM 8 ou superior
  • Mínimo de 200MB de espaço em disco
  • Conexão com internet (para comunicação com ThingSpeak)

Nota: Em caso de dúvidas ou dificuldades na instalação, entre em contato com a equipe de desenvolvimento.
